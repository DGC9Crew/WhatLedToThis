# "What let to this?"
A 2.5D platformer set in a post-apocalyptic future. Awakening after an unknown time in a bomb-bunker, you must go and find humans in an unrecognizable war-torn world. 

*Made with the jMonkeyEngine in Java.*

## Controls
- A: Move left
- D: Move right
- Space: Jump
- Esc: Quit Game
